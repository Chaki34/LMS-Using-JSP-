## 🚀 Project Overview

**LMS-Using-JSP** is a complete **Learning Management System (LMS)** built using  
☕ Java, 🌐 JSP, 🔧 Servlets, and 🗄️ MySQL.

It provides a simple and efficient platform for managing:
- 👨‍🎓 Students
- 👨‍🏫 Teachers
- 📚 Courses
- 📝 Assignments
- 📊 Results

---

## ✨ Features

- 🔐 User Login System (Admin / Student / Teacher)
- 📚 Course Management
- 🧑‍🏫 Teacher Dashboard
- 🎓 Student Dashboard
- 📝 Assignment Upload & Submission
- 📊 Result Management
- 🗄️ MySQL Database Integration
- ⚡ Simple & Responsive UI
